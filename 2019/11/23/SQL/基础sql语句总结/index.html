发表于 2019-11-23 | 分类于 Mysql

SELECT语句

查询同表不同列的值

查询同表全列的数据

SELECT DISTINCT（去除重复数据）

去除不同列重复的数据

ORDER BY(对结果集进行排序)

ASC(默认升序)，DESC（降序）

ASC或者 DESC 只对它紧跟着的第一个列名有效，其他不受影响，仍然是默认的升序。

WHERE(过滤记录)

INSERT INTO (插入数据)

（1） 全表插入数据

（2）按列插入数据

UPDATE

更新表已经存在的记录

必须携带where语句

DELETE

删除表中的记录（行）
